# Динамічний вебдодаток — Практична робота №4

## Мета
Розробка динамічного вебдодатку з використанням Fetch API, LocalStorage, Chart.js, інтерактивних компонентів та анімацій.

## Функціонал
1. Завантаження даних користувачів через [JSONPlaceholder API](https://jsonplaceholder.typicode.com/users).  
2. Динамічне відображення користувачів у вигляді карток з hover-анімаціями.  
3. Пошук користувачів за ім'ям у реальному часі.  
4. Збереження обраної теми (світла/темна) та імені користувача через LocalStorage.  
5. Візуалізація активності користувачів через [Chart.js](https://www.chartjs.org/).  
6. Skeleton Loader під час завантаження даних.  
7. Форма з валідацією введення.  
8. Віджет погоди через OpenWeatherMap API.

## Технології
- HTML5, CSS3, JavaScript (ES6+)
- Fetch API для асинхронного отримання даних
- LocalStorage для збереження стану
- Chart.js для графіка
- Анімації CSS для плавного ефекту появи карток та тем
